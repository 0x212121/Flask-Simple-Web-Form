Cara pertama

  1. Klik KPC Folder > Computer Name yang terletak pada bagian kanan bawah layar desktop. Seperti terlihat pada gambar
  2. Kemudian akan terbuka command prompt. Scroll pada bagian paling atas. Di baris hostname terdapat computer name-nya yang akan digunakan untuk input pada form

Cara kedua

  1. Buka command prompt atau cmd dengan cara klik start di pojok kiri bawah desktop, lalu ketikkan cmd dan kemudian klik command prompt
  2. Lalu ketikkan "hostname" (tanpa tanda kutip) di command prompt
